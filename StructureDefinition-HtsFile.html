1.0 StructureDefinition-HtsFile

This element is represented in FHIR as a profile of DocumentReference.

Phenopackets FHIR Comments
uri DocumentReference.content.attachment.url  
description DocumentReference.content.attachment.title  
hts_format DocumentReference.type The values are bound to the HTS format value set.
genome_assembly Extension  
individual_to_sample_identifiers NA The sample ids in the HTS files can be referenced by adding an identifier to the Specimen resource. Therefore, this attribute will be excluded from this profile. TODO: look at this in more detail when building the examples.

1.0.1 Formal Views of Profile Content

The official URL for this profile is: 

http://ga4gh.org/fhir/phenopackets/StructureDefinition/HtsFile

Phenopackets can be used to hold phenotypic information that can inform the analysis of sequencing data in VCF format as well as other high-throughput sequencing (HTS) or other data types.

This profile builds on DocumentReference.

This profile was published on Fri Oct 18 20:32:59 AEST 2019 as a draft by Global Alliance for Genomics and Health.
