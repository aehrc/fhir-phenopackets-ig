1.0 StructureDefinition-PedigreeNode

Mapping Notes

This element is represented in FHIR as a profile of FamilyMemberHistory-Genetic.

Phenopackets represents pedigrees using a very simple model based on the PED format. In this IG the family member history genetic profile is used as the basis to represent pedigree and extensions are used to add extra fields required to produce a PED file.

1.0.1 Formal Views of Profile Content

The official URL for this profile is: 

http://ga4gh.org/fhir/phenopackets/StructureDefinition/PedigreeNode

The Phenopackets Pedigree element represents a PED file, which describes the family relationships of each specimen along with their gender and phenotype. PED files are typically used by software for genetic linkage analysis. PED files have six mandatory columns, and additional optional columns (that are typically not used in the context of genomic analysis). FHIR already has resources and extensions that represent a pedigree. A pedigree node extends the FamilyMemberHistory-Genetic profile.

This profile builds on Family member history for genetics analysis.

This profile was published on Fri Nov 29 14:40:16 AEST 2019 as a draft by Global Alliance for Genomics and Health.
