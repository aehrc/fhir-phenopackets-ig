Extension: ExternalReference

This element encodes information about an external reference.

The official URL for this extension is: 

http://ga4gh.org/fhir/phenopackets/StructureDefinition/ExternalReference

Status: active
Extension maintained by: Alejandro Metke

Context of Use

This extension may be used on the following element(s):

  • {"type"=>"Extension"}
