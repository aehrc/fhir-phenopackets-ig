1.0 StructureDefinition-CancerStageGroup

This is an abstract class.

1.0.1 Formal Views of Profile Content

The official URL for this profile is: 

http://ga4gh.org/fhir/phenopackets/StructureDefinition/CancerStageGroup

A panel that contains cancer staging information. This class is the parent of classes containing information related to clinical and pathologic staging, including TNMClinicalStageGroup and TNMPathologicStageGroup.

This profile builds on Observation.

This profile was published on Fri Oct 18 20:32:59 AEST 2019 as a draft by Global Alliance for Genomics and Health.
